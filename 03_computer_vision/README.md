# 🖼️ Computer Vision

## 📌 Objective

This section focuses on building a strong foundation in Computer Vision using Deep Learning. It covers how computers interpret and analyze visual data such as images, along with the architectures and techniques used for image classification and related tasks.

The notebooks progress from understanding Convolutional Neural Networks (CNNs) to applying transfer learning and fine-tuning pre-trained models for real-world computer vision applications.
